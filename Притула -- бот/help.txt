Привет, Саша! Это инструкция по работе с ботом. Соблюдай её и мы быстро отладим бота, а затем он будет работать вечность.

В архиве помимо этого файла есть ещё 2. main.exe и text.txt. я настройка бота полностью находится в text.txt. 

Бот выполняет следующие функции:

1) По команде даёт воскрешение и очищение
Команды следующие:
/барыга огонь -- очищение огнём
/барыга очищение -- чистка от проклятий
/барыга воскреси -- воскрешение
Всё это связано со строкой "чаты для баффов" в text.txt

2) Автопост. Завязан на пяти строках:

-- Для общего чата "Общий чат", "Сообщение для общего" и "Ценники для общего". О настройке чуть позже.
-- Для архива "сообщение для архива" и "Архив"

3) Выдачу воды по команде "/барыга хочу Х воды". Работает в чатах с баффами.

4) Автооплату. С ней связаны те же строки, что и с общим чатом. Работает тольк в общем чате.

А теперь передём к настройке.

1) На сайте vk.ghost.io тебе необходимо раздобыть свой токен. Для работы с этим ботом нужен messages и в любое время. Больше он тут ничего не использует.
Токен необходимо вставить в первую строку. И нет, "|" нельзя удалять, он разделитель для строк в файле. Без него всё сломается.
2) Тебе необходимо раздобыть номера для чатов. В браузере будет примерно вот такая ссылка:
https://vk.com/im?peers=c305_-182985865_c307_267369151_435369438_c306_c231_c297_c149_391472391_c304_-206661168_c266_c265_c294_867555356&sel=c229
Тебе из неё нужно взять только последние цифры. У меня это 229. 
Сначала идёт общий чат, потом архив, затем все чаты с баффами.  
3) Необходимо написать два различных текста для чатов. Они подписаны! И автооплата подсасывает не из того же места, что и сообщение для общего чата! Автооплату настроим ниже.
4) И вот теперь мы должны настроить автооплату. По строчке вбиваем каждый ценник. Важный момент, учитывается количество золота и осколков. Для примера:
Пещерный корень - 150 золота
Отсюда бот подсасывает данные для автооплаты. И эта строка не уходит в общий чат, понятное дело.
5) Каждая ошибка, которая перезапустит бота, будет высвечиваться в логах ошибок. Я не стал подключать на всё попало, кое-что я поправил, будем смотреть. Через сутки этот лог и общие впечатления мне. Я буду смотреть по ситуации.

Надеюсь, всё будет хорошо и мы сработаемся!
